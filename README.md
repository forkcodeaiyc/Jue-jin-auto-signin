### 自动签到
- 自动模拟手动点击签到、免费抽奖
- 不会发生任何网络行为
- 每天启动浏览器时, 在后台打开标签页自动进行 
- 完毕后自动关闭标签页，不再执行

### 使用方法

1. git clone 本仓库
2. 打开浏览器扩展发者模式
3. 浏览器加载已解压的扩展程序
4. 完成
5. 可能会反复通知处于开发者模式 ， 可关闭浏览器提醒

### 流程和桌面通知
|流程|桌面通知|
|---|---|
|判断今日是否已执行|否|
| 判断是否登录| 未登陆通知 |
|判断是否已签到| 否  |
|  进行签到|  通知获取的矿石 |
| 判断是否已经抽奖| 否  |
| 进行免费抽奖|通知结果|
|  记录今日已执行|  否 |
| 关闭 tab|否|
